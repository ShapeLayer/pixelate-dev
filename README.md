# Pixelate Development Workspace

This repository is the development environment for `me.jonghyeon.pixelate`.

## Repository Strategy

Use two repositories:

1. `pixelate-dev` (this repo)
   - Full Unity project (Assets, ProjectSettings, Packages).
   - Used for implementation, testing, sample scene iteration.
   - Contains embedded package source at `Packages/me.jonghyeon.pixelate`.

2. [`pixelate`](https://github.com/shapelayer/pixelate) (Package)
